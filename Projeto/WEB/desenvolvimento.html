Desenvolvimento Web

No Primeiro semestre aprendemos a programar web em html5, css e javascript utilizamos o VSCode como aplicativo

HTML (Hypertext Markup Language):HTML, ou Linguagem de Marcação de Hipertexto, é a espinha dorsal da web. Ele fornece a estrutura básica para criar páginas da web, permitindo definir o conteúdo e a organização do documento. Por meio de tags (ou elementos), como html, head, body, div, p, h1, entre outras, podemos criar diferentes tipos de elementos, como parágrafos, títulos, imagens, links e listas. O HTML é uma linguagem de marcação bastante simples e fácil de aprender, sendo essencial para qualquer desenvolvedor web.

CSS (Cascading Style Sheets): CSS, ou Folhas de Estilo em Cascata, é uma linguagem utilizada para estilizar documentos HTML e controlar sua apresentação visual. Com o CSS, podemos definir cores, fontes, espaçamentos, tamanhos e layouts para os elementos HTML, tornando as páginas da web mais atraentes e consistentes. Utilizando seletores e propriedades, podemos aplicar estilos de forma seletiva a diferentes partes do documento, garantindo uma aparência visual coesa e agradável para os usuários. O CSS é uma parte fundamental do desenvolvimento web moderno, permitindo criar designs responsivos e esteticamente atraentes.

JavaScript: JavaScript é uma linguagem de programação de alto nível, interpretada e dinâmica, amplamente utilizada para criar interatividade em páginas da web. Com o JavaScript, podemos adicionar comportamentos dinâmicos às nossas páginas, como validação de formulários, animações, manipulação do DOM (Document Object Model), requisições assíncronas (AJAX), entre outros. Ele é suportado por todos os principais navegadores web e possui uma vasta biblioteca de frameworks e ferramentas para facilitar o desenvolvimento. JavaScript é uma das linguagens mais importantes no desenvolvimento web moderno, permitindo criar experiências interativas e ricas para os usuários.

tambem aprendemos a arvore DOM

A Árvore DOM (Document Object Model) é uma estrutura de árvore hierárquica que representa a estrutura de um documento HTML ou XML. Cada elemento na página web, como tags HTML, texto e atributos, é representado por um nó na árvore DOM. A estrutura é organizada de forma que cada nó tenha um pai e possa ter zero ou mais filhos, formando uma hierarquia.

Através da Árvore DOM, os navegadores web conseguem interpretar e manipular o conteúdo de uma página web. Ela é essencial para o desenvolvimento de aplicações web dinâmicas, pois permite que os desenvolvedores acessem e modifiquem os elementos da página usando linguagens de programação como JavaScript.

Em resumo, a Árvore DOM é uma representação estruturada e hierárquica do conteúdo de um documento HTML ou XML, sendo fundamental para a interatividade e manipulação dinâmica de páginas web
